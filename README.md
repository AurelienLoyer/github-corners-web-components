# Github 🐙 Corners 📐 Web Components 🥑

# 🦄

```html
<script src="https://unpkg.com/vue"></script>
<script src="../github-corners.min.js"></script>

<github-corners side="right" href="https://github.com/T3kstiil3"></github-corners>

```
       
# 🛠
